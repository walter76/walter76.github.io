    How to Work on a Branch with GitHub

    less than 1 minute read

    1. Create the remote branch on github
    2. Create a branch in the local repo
    3. Set the remote branch as upstream branch with git branch --set-upstream-to=origin/source source. You can now use git pull/push to synchronize your work.
    4. You can also pull/push the remote branch with git pull origin source
    5. Switch branches with git checkout <branch>
    6. List branches with git branch -v
    7. Create branch with git branch <branch>
